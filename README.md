# chat-system
Linux畅聊系统
client模块:畅聊系统client上层逻辑，提供client底层通信
server模块：畅聊系统server上层逻辑，提供server底层通信
comm模块：畅聊系统底层公共逻辑，如数据序列化。
window模块：client窗口模块
lib第三方库模块：提供第三方库模块
plug插件起停服务脚本模块：起停服务脚本

jsoncpp：提供基本数据序列化和反序列化功能；
基于ncurse/ncursew, 提供了client的窗口功能。
